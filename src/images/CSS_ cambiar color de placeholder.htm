CSS: cambiar color de placeholder

24 Feb 2014

Formas para cambiar el color (u otro estilo que le queramos dar) al atributo «placeholder» de los input o textarea de nuestros formularios. Por defecto observamos que es de un gris claro.

Para cambiar los estilos generales de «todos» los placeholder (ej. color de texto azul) se usa el pseudo-elemento ::placeholder:

::placeholder { color: blue; }
::placeholder { color: blue; }
::placeholder { color: blue; }

(*) Actualmente no tiene soporte para IE, pero lo puedes usar con el prefijo:

:-ms-input-placeholder { color: blue; } /* IE 10+ */
:-ms-input-placeholder { color: blue; } /* IE 10+ */
:-ms-input-placeholder { color: blue; } /* IE 10+ */

Y si usas algún navegador con alguna versión antigua, estos son los de webkit y moz:

::-webkit-input-placeholder { color: blue; } /* WebKit */
::-moz-placeholder { color: blue; } /* Firefox 19+ */
::-webkit-input-placeholder { color: blue; } /* WebKit */ ::-moz-placeholder { color: blue; } /* Firefox 19+ */
::-webkit-input-placeholder { color: blue; } /* WebKit */
::-moz-placeholder { color: blue; } /* Firefox 19+ */

Y para cambiar los estilos de los placeholder de una clase en concreto (ej. color rojo y negrita):

.mi-input::placeholder { color: red; font-weight: bold; }
.mi-input::placeholder { color: red; font-weight: bold; }
.mi-input::placeholder { color: red; font-weight: bold; }
